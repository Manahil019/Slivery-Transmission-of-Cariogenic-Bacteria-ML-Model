Caries Salivary Transmission Model analyzes behavioral and environmental factors driving salivary transmission of cariogenic bacteria (Streptococcus mutans) causing dental caries in Pakistan (>50% prevalence). This BDS and BME collaboration uses the Sindh Dental Caries Survey Dataset from Kaggle https://www.kaggle.com/datasets/manahilsiddique/salivary-transmission-of-cariogenic-bacteria/data(download here) with 300+ responses across 21 features preprocessed to 118D via ColumnTransformer. Six ML models benchmarked with MLP achieving 83.9% accuracy, 37% sensitivity, 82.8% specificity. Revealing 32% utensil sharing and 18% toothbrush sharing as top risks.
